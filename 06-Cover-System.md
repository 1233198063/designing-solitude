# 06 · Cover System — 封面系统 / Cover System

> 封面是读者决定要不要点进来的第一道门。好的封面系统让所有文章在视觉上属于同一个家族，同时每一篇又有自己的个性。
> The cover is the first gate that determines whether a reader clicks. A good cover system makes all articles visually part of the same family, while each piece retains its own personality.

## 封面结构 / Cover Anatomy

一个封面由哪些元素构成？每个元素的优先级和作用：
*What elements make up a cover? Priority and role of each:*

| 元素 / Element | 优先级 / Priority | 作用 / Role |
|--------------|-----------------|------------|
| 标题 / Title | P0 | 传达核心信息 |
| 副标题 / Subtitle | P1 | 补充说明 |
| 图像/背景 / Image | P1 | 建立情绪 |
| 品牌标识 / Brand Mark | P2 | 建立识别度 |

## 封面颜色规则 / Color Rules

> 封面颜色必须从 Color System（`08-Color-System.md`）中选取。
> Cover colors must be drawn from the Color System.

- 背景色 / Background: 从主色系中选 / From primary palette
- 文字色 / Text: 确保对比度 ≥ 4.5:1 / Minimum contrast ratio 4.5:1
