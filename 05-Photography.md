## 选图标准 / Image Selection Criteria

在选择一张图片时，用这三个问题检验：
*When selecting an image, apply these three questions:*

1. **这张图是否安静？** / Does this image feel quiet?
2. **这张图是否有"呼吸感"？** / Does this image have "breathing space"?
3. **这张图是否符合品牌的情绪关键词？** / Does this image match the brand's emotional keywords?
