# Designing Solitude — Brand Book

> 一个关于独处的专栏，也是一个关于如何用 AI 把模糊的创作理念设计成有生命力的个人品牌的完整记录。

这个仓库是我的 **Brand OS（品牌操作系统）**。它不会在"项目完成"那天结束，而是随着专栏一起生长。

---

## 这是什么

我在今年开始真正享受独处，想把这个过程写下来。写着写着，发现需要一个品牌来承载它——不只是一个好看的 logo，而是一套完整的表达语言：视觉、文字、内容结构、与读者的关系。

这个仓库记录的正是这套语言被设计出来的全过程，借助 AI，从零开始。

---

## 仓库结构

```
Brand Foundation
├── 01-Why.md              # 为什么做这件事，起点与动机
├── 02-Core-Values.md      # 核心价值观，品牌的骨架
├── 03-Moodboard.md        # 情绪板，感觉先于逻辑
├── 04-Visual-Language.md  # 视觉语言总览

Visual System
├── 05-Photography.md      # 摄影风格与选图标准
├── 06-Cover-System.md     # 封面系统设计规范
├── 07-Typography.md       # 字体系统
├── 08-Color-System.md     # 色彩系统

Content & Voice
├── 09-Writing-Voice.md    # 写作语气与风格
├── 10-Content-Framework.md # 内容框架与选题方法

experiments/               # 设计实验，记录每一次尝试和取舍
ai/                        # AI 工作流，提示词库与分析记录
assets/                    # 图片、字体等静态资源
```
