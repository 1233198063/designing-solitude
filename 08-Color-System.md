# 08 · Color System — 色彩系统 / Color System

> 颜色是情绪的速记符号。一套好的色彩系统应该让人一眼就感受到品牌的气质，同时足够灵活，能够适应不同的内容场景。
> Color is shorthand for emotion. A good color system should convey brand character at a glance, while remaining flexible enough for different content contexts.

---

## 无障碍对比度 / Accessibility Contrast

所有文字和背景的对比度必须满足：
*All text/background combinations must meet:*

- 正文 / Body text: ≥ **4.5 : 1** (WCAG AA)
- 大标题 / Large text: ≥ **3 : 1** (WCAG AA)

检查工具 / Checker: https://coolors.co/contrast-checker
