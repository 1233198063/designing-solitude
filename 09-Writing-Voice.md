## 句子风格 / Sentence Style

- **长度 / Length:** 短句优先，长句有节奏感 / Short sentences preferred; long ones with rhythm
- **人称 / Person:** 第一人称"我" / First person "I"
- **标点 / Punctuation:** 少用感叹号 / Use exclamation marks sparingly
- **过渡 / Transitions:** 自然，不用"首先、其次、最后" / Natural, avoid "firstly, secondly, finally"
