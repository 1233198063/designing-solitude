## 品牌分析 / Brand Analysis

### 提取视觉情绪 / Extract Visual Mood

```
我正在为一个关于独处的个人品牌分析参考图片。
请分析这张图片的：
1. 色彩情绪（用 3 个形容词）
2. 构图特点
3. 光线质感
4. 这张图让你联想到的品牌关键词（3 个）
5. 这张图是否适合一个"安静、真实、有深度"的个人品牌？原因是什么？

---

I'm analyzing reference images for a personal brand centered on solitude.
Please analyze this image for:
1. Color mood (3 adjectives)
2. Compositional characteristics
3. Quality of light
4. Brand keywords this image evokes (3)
5. Is this image appropriate for a brand that feels "quiet, authentic, and deep"? Why or why not?
```

### 标题优化 / Title Refinement

```
为以下文章生成 5 个标题选项。
文章主题：[主题]
品牌关键词：安静、独处、真实
目标读者：[描述]
要求：标题不超过 20 字，不用感叹号，不用"终于""一定""必须"等强推销词。

---

Generate 5 title options for the following article.
Article topic: [topic]
Brand keywords: quiet, solitude, authentic
Target reader: [description]
Requirements: under 20 characters (Chinese) / 8 words (English), no exclamation marks, no hard-sell language.
```

## 视觉设计 / Visual Design

### 封面描述生成 / Cover Description for AI Image Generation

```
我需要为一篇关于"[主题]"的文章生成封面图。
品牌视觉风格：低饱和度、自然光、大量留白、安静感
请生成一段英文 prompt，用于 Midjourney 或 DALL-E 生成封面图。
要求：避免人物面部特写，避免拥挤构图，偏向静物或场景。

---

I need a cover image for an article about "[topic]."
Brand visual style: desaturated, natural light, generous negative space, quiet atmosphere
Please generate an English prompt for Midjourney or DALL-E.
Requirements: avoid close-up faces, avoid cluttered compositions, favor still life or environmental scenes.
```
