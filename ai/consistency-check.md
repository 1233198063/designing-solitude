# AI · Consistency Check — 一致性审查 / Consistency Check

> 定期用 AI 对已发布内容做整体审查，发现偏离品牌方向的地方，及时校正。
> Periodically use AI to audit published content holistically, catch brand drift, and recalibrate.

---

## 何时做审查 / When to Run

- [ ] 每发布 5 篇文章后 / After every 5 articles
- [ ] 更新视觉系统后 / After updating the visual system
- [ ] 有新平台时 / When launching on a new platform
- [ ] 感觉"哪里不对"但说不清楚时 / When something feels off but you can't name it

---

## 审查维度 / Review Dimensions

### 1. 视觉一致性 / Visual Consistency

- 封面风格是否统一？ / Are cover styles consistent?
- 用色是否在规范内？ / Are colors within spec?
- 字体使用是否正确？ / Is typography applied correctly?

### 2. 语气一致性 / Voice Consistency

- 不同文章读起来像同一个人吗？ / Do articles across issues sound like the same person?
- 有没有某篇明显"跑调"？ / Is there an outlier that sounds off?

### 3. 内容方向一致性 / Content Direction

- 发布的内容是否都和内容支柱有关？ / Do all published pieces connect to a content pillar?
- 某个支柱是否被过度使用或忽视？ / Is any pillar over- or under-represented?
