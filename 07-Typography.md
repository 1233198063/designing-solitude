# 07 · Typography — 字体系统 / Typography

> 字体是品牌声音的视觉形态。中文字体的选择尤其关键，它直接影响内容的阅读体验和情绪基调。
> Typography is the visual form of a brand's voice. Chinese font selection is especially critical — it directly shapes reading experience and emotional tone.

---

## 字体选择原则 / Selection Principles

在选择字体前，先回答：
*Before choosing fonts, answer:*

- 这个品牌的阅读场景主要是什么？（手机 / 电脑 / 印刷）
  *Primary reading context? (Mobile / Desktop / Print)*
- 文字的情绪应该是什么？（正式 / 亲切 / 严肃 / 轻盈）
  *Desired emotional quality? (Formal / Warm / Serious / Light)*

## 字号规范 / Type Scale

基准字号（正文）/ Base size (body): **16px**

| 级别 / Level | 字号 / Size | 使用场景 / Usage |
|------------|-----------|----------------|
| xs | 12px | 标注、版权信息 / Labels, captions |
| sm | 14px | 辅助文字 / Secondary text |
| base | 16px | 正文 / Body |
| lg | 20px | 导语 / Lead paragraph |
| xl | 24px | 小标题 / Subheading |
| 2xl | 32px | 标题 / Heading |
| 3xl | 48px | 大标题 / Display |

---

## 排版规则 / Typography Rules

- [ ] 中英文之间加空格 / Space between Chinese and Latin characters
- [ ] 标点不出现在行首 / No punctuation at line start
- [ ] 正文行宽不超过 70 字符 / Body line length: max 70 chars
- [ ] 标题首字母大写（英文）/ Title case for English headings
